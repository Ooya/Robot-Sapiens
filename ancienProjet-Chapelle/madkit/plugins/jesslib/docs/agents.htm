Types of agents

Three types of agents are provided with the JessAgentLib library.

JessAgent

JessAgent are simple JessAgents that embed a Jess engine. They load an init file associated to their "loadFile" property when launched. JessAgent may be monitored by a JessMonitor. JessAgents are provided to explain how one can build its own JessAgent and embed a JessEngine into a Madkit Agent.

JessMonitor

A JessMonitor is an agent which DOES NOT embed a Jess engine, but may monitors any JessAgent whose AgentAddress is placed in its recipient property. The recipient property can be modified using the property box of the G-box. This agent is used for controlling simple JessAgents using ControlMessages.

EditJessAgent

EditJessAgents are special kind of JessAgents that are combined with an editor:

Inputs from the top buffer may be evaluated. Their results is displayed in the bottom editing area.

Run

Run the Jess engine associated to the agent.

Halt

Stops the Jess engine.

Facts

Displays the list of facts.

Rules

Displays the list of rules.

Reset

Reset the Jess engine, deleting all facts and asserting an initial-fact.

Re-init

Clear the Jess engine, deleting all rules, defined functions and facts. Then, loads again the file associated to the "loadFile" field displayed in the property editor.

Watch

Opens a dialog

Eval selection

Evaluate the selection from the input area (top editor) and returns the result in the output area (bottom editor).

Eval buffer

Evaluate the whole input area (top editor) and returns the result in the output area (bottom editor).

Load

Open a dialog box to load a file and evaluate it.

Open

Open dialog box to open a file in the top editor

Save

Save the content of the top editor into a text file.
